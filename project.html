Project 2: Optimazation Problem with Uncertainty
In the distributed framework, we use the scenario theory to deal with the uncertainty occuring in the optimization problem. We first extract some samples of uncertain variable, and then convert the uncertain optimization problem into a determined optimization problem. Now, we use the probability tools to evalute whether the optimal solution of the determined problem is feasible for the original problem . Finally, we use a distributed algorithm to solve the determined problem.

Reference: Kostas Margellos, Alessandro Falsone, Simone Garatti, Maria Prandini, Distributed Constrained Optimization and Consensus in Uncertain Networks via Proximal Minimization, IEEE Trans. Automat. Controlvol. 63, no. 5, pp. 1372-1387, May, 2018. [paper and code]
